# sentry.io
Sentry.io website

## Setup
```
$ bundle install
$ npm install -g grunt-cli
$ npm install
```
### Previewing/Development
Starts the BrowserSync on `localhost:3000`

`$ grunt`

### Building
Will build the site, putting files in the `dist` directory.

`$ grunt build`

### Deploying
Will build and deploy to gh-pages.

`$ grunt deploy`
